Tic Tac Toe AI

This is a project that I made to learn how to code. It is a website that I made using HTML and CSS.

This was my first ever python project (and first project in general!), and it made me learn a lot about python but also how programming works, concepts that make the foundation for every programming language. Concepts that made the learning of other languages 10x easier.

But speaking of the project itself, it is a TicTacToe artificial intelligence (as you probably guessed already), which contains 3 difficulties:

  • Easy - the AI chooses randomly. Rather impossible to lose
  • Medium - the AI isn’t too smart but will try its best to win anyways
  • Impossible - the AI is impossible to beat. You will be able to tie, but no way you will win

Wikipedia Randomizer

This is probably the easiest project in this list, but for me it marks the moment i begun using APIs, and it changed everything: the possibility to interact with thousands of services such as Wikipedia, Spotify, Youtube, Google Docs, and the list goes on.

But getting to the juice, the project itself is really easy: you insert a language and an optional input, and the program will search the optional input on wikipedia (or take a random article if no input was given) and from there the program gets a random link from that wikipedia article and repeats.

The thing that fascinates me about this is that in the sequence of random articles you see articles that are correlated, but the correlation goes away quickly.
For example, an example run starting with Roosevelt leads to Bully pulpit, Conservative coalition, Justified (season 5) and so on.

Offline Reddit

This is the first project where i used some frontend (html, css, javascript), and it both marked a new beginning for my programming journey as i learned new languages, but it also came from a necessity: in the period i made this, i had to be in a flight, and didn’t want to give up those 2 hours of fly time to reading, sleeping or anything like that: i wanted to browse reddit, but of course there was no internet connection.

So, I made this, which takes some posts and it’s comments while you have internet access, and saves them in a file. Then, when you need it, you can easily load them in a UI that is very similar to Reddit’s.

Even though i still spent the whole flight sleeping, I am still proud of my project.
